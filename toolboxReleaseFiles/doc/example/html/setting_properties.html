Setting graphic object properties

There exists two ways to set graphic object properties depending on how often the property needs to be set.
  • setOnce allows to set the property once during the initialization phase (1st iteration).
  • setIter allows to update a property at every iteration
Both instructions have a same usage. They requires to pass a handle. This handle is typically sent by a graphic function (plot, surf, title, etc).

Setting a property only once

clearvars
close all
clc
 
id=iterativeDisplay;
 
% If the following line is commented, then the title color will not change,
% only the text will change
id.disableOptimization('title');
 
hFig=id.figure;
set(hFig,'visible','on'); % Undock the figure from the live editor
 
x=linspace(-10,10,20);y=x.^2;
for i=1:10
if i<10
id.newIteration;
else
id.finalIteration;
end
id.plot(x,y*i);
% Display a title and save a handle
h=id.title('red');
% Modify the property once
id.setOnce(h,'Color','r');
drawnow;
pause(0.1)
end

Setting a property at every iteration

clearvars
close all
clc
 
id=iterativeDisplay;
 
% If the following line is commented, then the title color will not change,
% only the text will change
id.disableOptimization('title');
 
hFig=id.figure;
set(hFig,'visible','on'); % Undock the figure from the live editor
 
x=linspace(-10,10,20);y=x.^2;
for i=1:10
if i<10
id.newIteration;
else
id.finalIteration;
end
id.plot(x,y*i);
% Display a title and save a handle
h=id.xlabel('This is a growing label');
% Modify the property at every iteration
id.setIter(h,'FontSize',10+i);
drawnow;
pause(0.1)
end
