Iterative Display Toolbox

If you need to visualize data within an iterative algorithm, then the Iterative Display Toolbox can do it faster than classical Matlab operation without the hassle of managing yourself the handles to all the graphics object.

Finally, here is an example script. Basically, you just have to add "id." before using Matlab's command and you need to add a call to newIteration or finalIteration at the begining of the loop.
id=iterativeDisplay();
n=100; % Number of iterations
 
for i=1:n
% Add this piece of code at the begining of your main loop
if i<n
% Start a new iteration
id.newIteration;
else
% Last iteration needs to be indicated
id.finalIteration;
end
 
id.figure;
id.plot(rand(1,10)); % Plot some data
id.grid('on');
id.xlabel('Time');
id.ylabel('data');
id.title('Iterative Display Demo');
drawnow; % Force display update
end
